Test Infrastructure Improvements

  • Standardised ts-jest with isolatedModules: true across all 5 module Jest configs.
  • Added centralised mock setup files for Azure Key Vault, Twilio, SendGrid, FCM (firebase-admin), Service Bus, SocketIO, Redis (ioredis).
  • Switched coverage provider to v8 with --runInBand for stable runs on Node 24 + jest 29.
  • Established consistent singleton-reset pattern: (Service as any).instance = null in afterEach.
